## Install from the AUR

Use your favorite AUR helper:

```bash
yay -S t3code-bin
# or
paru -S t3code-bin
```

## What this package does

- Tracks upstream T3 Code releases.
- Packages the upstream x86_64 AppImage.
- Publishes updates to the AUR automatically.
